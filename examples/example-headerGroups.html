Demonstrates:

  • HeaderGroups plugin

Instructions:

같은 그룹으로 설정된 컬럼의 헤더 위에 그룹명을 표시한다.

Usage:

var columns = [
  {id: "title", name: "Title", field: "title"},
  {id: "duration", name: "Duration", field: "duration", group:"Group 1"},
  {id: "%", name: "% Complete", field: "percentComplete", group:"Group 1"},
  {id: "start", name: "Start", field: "start", group: "기 간"},
  {id: "finish", name: "Finish", field: "finish", group: "기 간"},
  {id: "effort-driven", name: "Effort Driven", field: "effortDriven"}
];

plugin = new Slick.Plugins.HeaderGroups();
grid.registerPlugin(plugin);
